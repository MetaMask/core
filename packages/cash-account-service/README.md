# `@metamask/cash-account-service`

Cash account service.

This service provides operations for creating and managing Cash accounts derived from HD keyrings.

## Installation

`yarn add @metamask/cash-account-service`

or

`npm install @metamask/cash-account-service`

## Contributing

This package is part of a monorepo. Instructions for contributing can be found in the [monorepo README](https://github.com/MetaMask/core#readme).
